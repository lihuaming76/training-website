# 步骤3 预约页面 — 精修指令（基于最新截图）

> **背景：** 步骤3页面已完成主要布局重构，但仍有一屏显示不全、左右两栏错位、间距偏松等问题。以下按优先级排列，请严格按顺序执行。

---

## P0：一屏显示（这个问题必须彻底解决）

### 1. 右栏卡片内部间距全面压缩

当前右栏从"选择您方便的沟通方式"到"预约电话时段"按钮的纵向链路太长，导致底部内容溢出视口。对右栏卡片内部做一次彻底的间距压缩：

```
选择您方便的沟通方式    ← 标题
    ↓ 12px
专家信息条（头像+姓名）  
    ↓ 12px
二维码（140px × 140px） ← 确认是140px，如果仍大于140请强制设置
    ↓ 8px
扫码说明文字 + 微信号
    ↓ 10px
—— 或 ——
    ↓ 10px
[预约电话时段] 按钮
    ↓ 8px
社会证明小字
```

具体CSS：
```css
/* 专家信息条与二维码之间 */
margin-bottom: 12px !important;

/* 二维码图片 — 强制140px */
width: 140px !important;
height: 140px !important;

/* 二维码到下方文字 */
margin-bottom: 8px !important;

/* "或"分隔线 — 上下margin压缩 */
margin-top: 10px !important;
margin-bottom: 10px !important;

/* 右栏卡片整体内边距压缩 */
padding: 20px 24px !important;
```

### 2. 主标题区与两栏内容之间的间距压缩

"帮您定位卡点、理清下一步该做什么" 到下方两栏内容区之间当前约 40-50px，压缩到 20px：

```css
/* 副标题 */
margin-bottom: 20px !important;
```

### 3. 左右两栏顶部对齐

当前左栏 "这15分钟，您会得到：" 的起始位置明显低于右栏卡片顶部，存在纵向错位。

```css
/* 两栏网格容器 */
align-items: start !important;
```

确保左栏第一行文字和右栏卡片的上边缘在同一水平线上。

### 4. 锁定页面一屏

```css
/* 页面最外层容器（桌面端） */
height: 100vh;
overflow: hidden;
```

## P1：视觉层级修正

### 5. "扫码立即开始诊断" 移到二维码下方

当前这行红色文字在二维码**上方**，与专家信息条争夺注意力，且"立即开始"的语气暗示点击动作，容易误导。

**改动：**
- 将这行文字从二维码上方**移到二维码下方**
- 文案改为：`扫码添加微信，预约15分钟专属诊断`
- 颜色从红色改为 `#E8C547`（accent黄色）
- 字号 `14px`，`font-weight: 600`

改动后右栏的视觉层级变为：
```
标题 → 专家背书 → 二维码（视觉焦点）→ 行动说明 → 微信号 → 或 → 电话预约
```
从上到下，逻辑通顺。

### 6. 序号圆圈颜色统一为accent黄色

当前三个序号①②③的圆圈是**绿色**，但页面主accent色是黄色，绿色在这里没有语义理由（不是"完成"状态）。

```css
/* 序号圆圈 */
background: rgba(232, 197, 71, 0.15) !important;
color: #E8C547 !important;
```

### 7. 专家头像占位符融入主题

当前"李"字的圆形占位符背景色偏亮（绿色），在深色主题中突兀。

```css
/* 头像占位圆 */
background: #2A2A2A !important;
color: #E8C547 !important;
border: 1px solid rgba(255, 255, 255, 0.1) !important;
```

上线前替换为真实照片。

### 8. "不推销，只给专业判断" 加分隔线

当前这句话紧贴第3个价值点下方，看起来像第4个价值点。需要与上方内容做视觉分隔：

```css
margin-top: 20px !important;
padding-top: 16px !important;
border-top: 1px solid rgba(255, 255, 255, 0.06) !important;
```
