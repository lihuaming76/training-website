# 李华明·数据与AI驱动企业经营培训网站

一个精心设计的B2B培训服务网站，专注于数据与AI驱动的企业经营培训课程推广。

## 📋 项目概述

这是一个纯前端静态网站，采用现代化设计风格，包含完整的互动式咨询流程，帮助潜在客户从浏览到预约15分钟咨询的完整转化路径。

### 核心特色

- **4步渐进式互动流程** - 网站的核心功能，引导用户完成需求诊断
- **精致商务设计** - 深海军蓝 × 金色的高端配色方案
- **玻璃拟态效果** - 现代化的视觉设计语言
- **完全响应式** - 完美适配桌面、平板、移动设备
- **纯静态实现** - 无需后端，可直接部署到任何静态托管服务

## 🚀 快速开始

### 方法1：直接打开（最简单）

1. 下载所有文件到同一文件夹
2. 双击 `index.html` 文件
3. 网站将在默认浏览器中打开

### 方法2：本地服务器（推荐）

使用本地服务器可以避免某些浏览器的CORS限制：

**使用Python（如果已安装）：**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**使用Node.js（如果已安装）：**
```bash
# 安装http-server
npm install -g http-server

# 启动服务器
http-server -p 8000
```

**使用VS Code：**
- 安装 "Live Server" 扩展
- 右键点击 `index.html`
- 选择 "Open with Live Server"

然后在浏览器中访问：`http://localhost:8000`

## 🌐 部署到生产环境

### 部署到GitHub Pages

1. 创建GitHub仓库
2. 上传所有文件
3. 进入仓库设置 → Pages
4. 选择分支（通常是main）
5. 点击保存，等待部署完成

### 部署到Netlify

1. 访问 [netlify.com](https://www.netlify.com)
2. 拖拽整个文件夹到部署区域
3. 等待自动部署完成
4. 获得免费的HTTPS域名

### 部署到Vercel

1. 访问 [vercel.com](https://vercel.com)
2. 导入项目或拖拽文件夹
3. 自动部署并获得域名

### 部署到传统服务器

将所有文件上传到服务器的Web根目录（如 `/var/www/html/` 或 `/public_html/`）

## 🔧 自定义指南

### 修改颜色
编辑 `styles.css` 文件的 `:root` 部分：
```css
:root {
    --color-primary: #你的颜色;
    --color-secondary: #你的颜色;
}
```

### 修改内容
直接编辑 `index.html` 文件中的文本内容。

### 添加真实二维码
在 `index.html` 的第460-462行，替换二维码占位符：
```html
<div class="qr-code-placeholder">
    <img src="your-qrcode.png" alt="微信二维码" style="width: 256px; height: 256px;">
</div>
```

### 修改联系方式
在 `index.html` 的第465行和第914-916行修改微信号、邮箱、电话。

## 📊 性能优化建议

1. **图片优化**: 如果添加真实图片，建议使用WebP格式并压缩
2. **字体优化**: 考虑使用 `font-display: swap` 避免字体加载阻塞
3. **CDN加速**: 将Google Fonts替换为国内CDN（如字节跳动CDN）

## 🐛 常见问题

**Q: 为什么本地打开时某些功能不工作？**
A: 使用本地服务器而不是直接打开HTML文件。

**Q: 如何修改微信二维码？**
A: 替换 `index.html` 第460行的占位符为真实图片。

**Q: 移动端菜单无法关闭？**
A: 确保 `script.js` 已正确加载。

**Q: 轮播图不自动播放？**
A: 检查浏览器控制台是否有JavaScript错误。
