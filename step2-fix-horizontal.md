# 步骤2页面一屏显示 — 紧急修复指令

> 当前页面仍然无法一屏显示，根本原因是"您会带走"区域的三个交付物仍然是**纵向堆叠**，每个占一整行。必须改为横向三列。

---

## 第一步：找到"您会带走"下面的三个交付物容器

在代码中找到包含以下三个卡片的父容器：
- "一份认知地图"
- "一个方向选择"  
- "一个启动框架"

这个父容器当前的布局方式大概是 `flex-direction: column` 或者没有设置 grid，导致三个子元素纵向排列。

## 第二步：将父容器改为横向三列网格

对这个父容器，**强制设置**以下CSS（覆盖原有样式）：

```css
display: grid !important;
grid-template-columns: repeat(3, 1fr) !important;
gap: 14px !important;
flex-direction: unset !important;
```

## 第三步：压缩每个交付物卡片的高度

对每个交付物卡片（子元素），设置：

```css
padding: 14px 16px !important;
min-height: unset !important;
height: auto !important;
```

确保卡片内部的绿色对勾和文字是**水平排列**（对勾在左，文字在右），而不是上下排列：

```css
/* 卡片内部容器 */
display: flex !important;
align-items: flex-start !important;
gap: 10px !important;
flex-direction: row !important;
```

## 第四步：把底部资料卡片和CTA按钮放到同一行

找到"AI项目画布（样例）"、"落地断点自检表"这两个卡片和"预约工作坊席位→"按钮。

将它们包裹在一个 flex 容器中：

```css
display: flex !important;
align-items: stretch !important;
gap: 16px !important;
```

- 左侧：两张资料卡片并排（各占 `flex: 1`）
- 右侧：CTA按钮区域（`min-width: 240px`），包含主按钮和"返回"次按钮纵向排列

## 第五步：锁定页面高度

在步骤2页面的最外层容器上设置：

```css
height: 100vh;
overflow: hidden;
```
