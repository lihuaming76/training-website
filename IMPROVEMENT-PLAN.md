## Task 1：添加滚动入场动画（优先级最高）

### 目标
让页面滚动时各 section 的内容有节奏感地出现，提升现代感和高端感。

### 技术方案
- 使用 Intersection Observer API，不引入第三方库
- 元素进入视口 15% 时触发，仅触发一次（触发后 unobserve）
- 使用 `cubic-bezier(0.16, 1, 0.3, 1)` 缓动函数（弹性出场感）
- 必须添加 `@media (prefers-reduced-motion: reduce)` 无障碍支持

### 各区域动画规则

| 区域 | 动画类型 | 说明 |
|------|----------|------|
| **Hero 左侧** | 依次从下滑入（stagger） | 标题 → 承诺语 → 证据卡片 → 详情，每个延迟 150ms |
| **Hero 右侧** Trust Panel | 从右侧滑入 | 延迟 300ms，translateX(50px) → 0 |
| **诊断区** 标题 | 从下淡入 | 普通 fade-up |
| **诊断区** 3张卡片 | 交错入场（stagger） | 3张卡片依次出现，每张间隔 120ms |
| **解决流程** 整体卡片 | 从下淡入 | 普通 fade-up |
| **解决流程** 5步骤+4箭头 | 依次入场 | 9个子元素（step, arrow, step, arrow...）按 nth-child 顺序依次出现，间隔 150ms |
| **互动流程区** 标题 | 从下淡入 | 普通 fade-up |
| **成功案例** 标题 | 从下淡入 | 普通 fade-up |
| **成功案例** Tab栏 | 从下淡入 | 普通 fade-up |
| **成功案例** 内容 | 从下淡入 | 普通 fade-up |
| **课程对比** 标题 | 从下淡入 | 普通 fade-up |
| **课程对比** 表格 | 从下淡入 | 普通 fade-up |
| **交付物轮播** 标题 | 从下淡入 | 普通 fade-up |
| **交付物轮播** 轮播区 | 缩放入场 | scale(0.9) → scale(1) |
| **讲师介绍** 标题 | 从下淡入 | 普通 fade-up |
| **讲师介绍** 左侧照片 | 从左滑入 | translateX(-60px) → 0 |
| **讲师介绍** 右侧内容 | 从右滑入 | translateX(60px) → 0 |

### 注意事项
- 动画 class 和通用 reveal class 不要冲突（例如 process-steps 的子元素交错动画建议用独立 class 如 `reveal-process`，不要和父级的 reveal 混在一起）
- 所有动画持续时间 0.6s-0.9s，不要太快也不要太慢
- 确保在不同屏幕尺寸下动画都正常工作

## Task 4：增加 Section 背景节奏（深浅交替）

### 目标
当前大部分 section 背景是白色或浅灰，缺少视觉层次。通过深浅交替让页面有呼吸感。

### 建议配色方案

| Section | 当前背景 | 建议调整 |
|---------|---------|---------|
| Hero | 深海军蓝渐变 ✅ | 保持不变 |
| 诊断区 | 白色 | 改为极浅暖灰 `#F8F7F5` 或保持白色 |
| 解决流程 | 浅灰渐变 | 改为更深一度 `#F0EDE8`（暖色调）|
| 互动流程 | 白色 | **改为深色背景**，使用深海军蓝 `#0F172A` + 白色文字，作为核心转化区域突出 |
| 成功案例 | 默认 | 浅米色 `#FAF8F5` |
| 课程对比 | 默认 | 白色 |
| 交付物 | 默认 | 浅灰 `#F5F4F2` |
| 讲师介绍 | 白色 | 保持白色 |
| Footer | 深海军蓝 ✅ | 保持不变 |

### 关键要求
- 互动流程区改为深色背景是重点，因为这是核心转化区，需要和其他 section 形成强烈视觉反差
- 如果互动流程改为深色背景，其中的卡片、按钮、文字颜色都需要同步调整（卡片背景改为 `rgba(255,255,255,0.05)`，文字改为白色/浅色）
- section 之间可以考虑用微妙的分割线或渐变过渡，避免突兀跳变

```html
<p>&copy; 2025 李华明·数据与AI培训 | 保留所有权利</p>
```

```html
<p>&copy; <span id="currentYear"></span> 李华明·数据与AI培训 | 保留所有权利</p>
<script>document.getElementById('currentYear').textContent = new Date().getFullYear();</script>
```
