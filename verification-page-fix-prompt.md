# 实战验证页面 — 首屏布局修复指令

> **两个目标：** ① 释放顶部幽灵占位空间，显示被遮挡的标题文字；② 将所有内容（标题+Logo墙+案例标题+引言+向下查看提示）压缩到一屏内。

---

## 问题1：顶部幽灵占位空间（Phantom Spacer）

### 现象
页面顶部（导航栏下方、"主理人曾于 埃森哲/德勤/IBM..."文字上方）存在一大段空白区域，大约 100-150px 高度。这段空白挤掉了下方内容的空间，同时可能遮挡了一行标题文字（"23年顶尖咨询实战，服务过这些行业标杆"）。

### 排查方向
这个空白通常来自以下几种情况，请逐一检查：

1. **页面顶部有一个隐藏的 Hero/Banner 区域**，虽然内容不可见但仍占据高度：
```css
/* 查找类似这样的元素 */
.hero, .banner, .page-header, [class*="hero"], [class*="banner"] {
  /* 如果存在，设置 */
  display: none !important;
  /* 或 */
  height: 0 !important;
  min-height: 0 !important;
  padding: 0 !important;
  margin: 0 !important;
  overflow: hidden !important;
}
```

2. **导航栏是 fixed/sticky 定位，页面内容有对应的 padding-top 或 margin-top 来避让导航栏**，但数值设得过大：
```css
/* 查找主内容容器的 padding-top / margin-top */
main, .main, .content, .page-content, [class*="content"] {
  /* 如果 padding-top 远大于导航栏高度（约60-70px），减小到合理值 */
  padding-top: 70px !important; /* 仅比导航栏高度多10px即可 */
}
```

3. **存在一个空的占位 div**（可能是其他页面共用的组件，在这个页面没有内容但仍占空间）：
```bash
# 在浏览器 DevTools 中检查导航栏到第一行可见文字之间的所有DOM元素
# 找到高度>0但内容为空的元素，将其隐藏
```

### 执行
找到幽灵空间的来源后，将其高度归零或隐藏。确保页面标题"23年顶尖咨询实战，服务过这些行业标杆"紧跟在导航栏下方，中间只留约 **30-40px** 的正常间距。

## 问题2：首屏一屏显示

### 目标
在修复幽灵空间后，确保以下所有内容在首屏视口（100vh）内完整可见：

```
┌─────────────────────────────────────────────┐
│  导航栏                                       │  ~60px
├─────────────────────────────────────────────┤
│  23年顶尖咨询实战，服务过这些行业标杆            │  标题区 ~70px
│  主理人曾于 埃森哲/德勤/IBM...                  │
├─────────────────────────────────────────────┤
│  [Cola] [U] [ANTA] [Amer] [波司登] [...]     │  Logo墙 ~80px
├─────────────────────────────────────────────┤
│  26个行业标杆落地实录·验证3种破局路径            │
│  将顶级企业的试错经验...                        │  案例标题区 ~90px
├─────────────────────────────────────────────┤
│  " 23年里我参与了几十个转型项目...               │
│     ...现在都融入了我的工作坊。                  │  引言区 ~150px
│                    — 李华明 Pallas Studio 主理人 │
├─────────────────────────────────────────────┤
│              ↓ 向下查看 3 大实战拆解              │  ~50px
└─────────────────────────────────────────────┘
                                          总计 ~500-550px
```

### 间距压缩清单

如果修复幽灵空间后仍然放不下，按以下顺序逐步压缩：

```css
/* 1. 标题区与Logo墙之间 */
margin-bottom: 20px !important;  /* 如果当前更大 */

/* 2. Logo墙与案例标题之间 */
margin-bottom: 24px !important;

/* 3. 案例标题区与引言之间 */
margin-bottom: 20px !important;

/* 4. 引言区上下内边距 */
padding: 20px 40px !important;  /* 如果当前更大 */

/* 5. 引言区与"向下查看"之间 */
margin-bottom: 16px !important;

/* 6. Logo墙的上下内边距 */
padding: 16px 0 !important;

/* 7. "26个行业标杆..."标题字号如果过大，可微缩 */
font-size: 32px !important;  /* 确认当前值，如果>36px可缩小 */
```
