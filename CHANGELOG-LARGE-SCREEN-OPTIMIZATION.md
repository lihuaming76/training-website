# 首屏大屏优化变更清单

## 优化目标
针对 **2880×1800 且系统缩放 200%（等效视口约 1440×900）** 的大屏幕，优化首屏（Hero区域）使其不滚动即可完整看到核心信息与主CTA。

## 验收标准
✅ 1440×900 视口下，不滚动即可看到：
- 导航栏
- 主标题/副标题
- 左侧"适合谁/怎么做/拿到什么"摘要
- 三张结果卡片
- 主CTA按钮

---

## 关键改动点（10条核心优化）

### 1. Hero区域整体高度压缩 ⭐⭐⭐
**改动**：
- `min-height: 100vh` → `min-height: auto`（大屏下）
- `padding-top: 80px` → `padding-top: 80px`，`padding-bottom: 2rem` → `1.5rem`

**效果**：Hero区域不再强制占满整屏，根据内容自适应高度

---

### 2. 主标题字号大幅压缩 ⭐⭐⭐
**改动**：
- 主标题：`3.5rem` → `2.5rem`（大屏下）
- 副标题：`1.5rem` → `1.125rem`（大屏下）
- 标题间距：`gap: 1rem` → `0.5rem`，`margin-bottom: 2rem` → `1rem`

**效果**：标题区域高度减少约30%

---

### 3. 左侧引导条高度压缩 ⭐⭐
**改动**：
- `padding: 2rem` → `1rem 1.25rem`（大屏下）
- 字号：`1.25rem` → `0.9375rem`（大屏下）
- `margin-bottom: 2rem` → `1rem`

**效果**：引导条高度减少约40%

---

### 4. 采购三问摘要紧凑化 ⭐⭐
**改动**：
- `gap: 1rem` → `0.5rem`（大屏下）
- `padding: 1.5rem` → `0.875rem 1rem`（大屏下）
- 标签字号：`0.875rem` → `0.75rem`
- 内容字号：`0.95rem` → `0.875rem`

**效果**：摘要区域高度减少约35%

---

### 5. 三张卡片关键数字对比度强化 ⭐⭐⭐
**改动**：
- 关键数字使用 **深色底pill标签** 方案
- 颜色：`#F2C14E`（亮金色）
- 背景：`rgba(242, 193, 78, 0.18)`
- 边框：`1px solid rgba(242, 193, 78, 0.35)`
- 内边距：`0.125rem 0.5rem`

**效果**：关键数字（1个、4张A3、75%+等）一眼可扫到，对比度显著提升

---

### 6. 三张卡片间距压缩 ⭐⭐
**改动**：
- `gap: 1rem` → `0.75rem`（大屏下）
- `padding: 1.5rem` → `1rem`（大屏下）
- `margin-bottom: 2rem` → `1rem`
- 图标：`2rem` → `1.5rem`

**效果**：卡片区域高度减少约25%

---

### 7. 右侧卡片扁平化改造 ⭐⭐⭐
**改动**：
- **左右比例调整**：`1fr 1fr` → `60fr 40fr`（大屏下）
- **卡片宽度**：`max-width: 400px` → `max-width: 100%; width: 100%`
- **卡片高度**：`padding: 2.5rem` → `1.25rem 1.5rem`
- **三段式改为横向扁平列表**：`display: flex; align-items: baseline`
- **阴影更轻**：`box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06)`

**效果**：右侧卡片高度减少约40%，视觉更扁平轻量

---

### 8. 背书条轻量化 ⭐
**改动**：
- 背景：`var(--color-gold-light)` → `rgba(212, 175, 55, 0.08)`
- 新增边框：`1px solid rgba(212, 175, 55, 0.15)`
- `padding: 1rem` → `0.625rem 0.75rem`
- 字号：`0.8125rem` → `0.75rem`

**效果**：背书条更轻量，不占据过多视觉重量

---

### 9. 按钮高度降低 ⭐⭐
**改动**：
- `padding: 1rem 2rem` → `0.75rem 1.75rem`（大屏下）
- `font-size: 1rem` → `0.9375rem`（大屏下）
- 按钮间距：`gap: 1rem` → `0.75rem`

**效果**：按钮高度从约48px降到约44px

---

### 10. "怎么开始"三步流程压缩 ⭐
**改动**：
- `margin-top: 3rem` → `1.25rem`（大屏下）
- `padding: 1.5rem` → `0.875rem 1rem`（大屏下）
- 字号：`0.875rem` → `0.75rem`（大屏下）
- `gap: 1rem` → `0.75rem`

**效果**：三步流程高度减少约35%

---

## 技术实现方式

所有优化都通过 `@media (min-width: 1280px)` 媒体查询实现，确保：
- ✅ 只在大屏幕（≥1280px）生效
- ✅ 不影响平板和移动端的现有布局
- ✅ 保持响应式设计的完整性
